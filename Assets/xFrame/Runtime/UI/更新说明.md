# UI组件系统更新说明

## 🎉 v2.1 - 系统优化与增强 (2024-12-04)

### Bug修复
- ✅ 修复 `UITabContainer.OnUIDestroy()` 中错误调用 `OnDestroy()` 的问题，改为正确调用 `base.OnUIDestroy()`

### 新功能

#### 1. UI事件通知系统
UIManager现在会在UI打开/关闭时自动发送事件通知：
- `UIOpenedEvent` - UI打开时触发
- `UIClosedEvent` - UI关闭时触发
- `UILayerChangedEvent` - 层级UI数量变化时触发

```csharp
// 订阅UI打开事件
xFrameEventBus.SubscribeTo<UIOpenedEvent>(OnUIOpened);

private void OnUIOpened(ref UIOpenedEvent evt)
{
    Debug.Log($"UI打开: {evt.UIType.Name}, 层级: {evt.Layer}");
}
```

#### 2. 泛型UIComponent支持
新增 `UIComponent<TData>` 泛型基类，提供类型安全的数据设置：

```csharp
// 定义数据类型
public class ItemData
{
    public int Id;
    public string Name;
}

// 使用泛型组件
public class ItemComponent : UIComponent<ItemData>
{
    protected override void OnSetData(ItemData data)
    {
        // 类型安全，无需类型转换
        Debug.Log($"物品: {data.Name}");
    }
}

// 使用
itemComponent.SetData(new ItemData { Id = 1, Name = "剑" });
var data = itemComponent.GetData(); // 返回ItemData类型
```

#### 3. UIWindow遮罩系统完善
模态窗口现在自动创建和管理遮罩：
- 自动创建全屏遮罩
- 支持自定义遮罩颜色和透明度
- 支持点击遮罩关闭窗口
- 窗口关闭时自动销毁遮罩

```csharp
public class MyDialog : UIWindow
{
    public override bool IsModal => true;           // 模态窗口
    public override bool CloseOnMaskClick => true;  // 点击遮罩关闭
    public override float MaskAlpha => 0.7f;        // 遮罩透明度
    public override Color MaskColor => Color.black; // 遮罩颜色
}
```

#### 4. UI动画系统
新增 `IUIAnimation` 接口和预定义动画：

```csharp
// 使用预定义动画
UIAnimations.PopIn.Play(rectTransform);
UIAnimations.FadeIn.Play(rectTransform);
UIAnimations.SlideInFromBottom.Play(rectTransform);

// 异步播放
await UIAnimations.PopIn.PlayAsync(rectTransform);

// 自定义动画
var customAnim = new ScalePopAnimation(duration: 0.5f);
customAnim.Play(rectTransform, () => Debug.Log("动画完成"));
```

预定义动画：
- `PopIn` / `PopOut` - 缩放弹出/收缩
- `FadeIn` / `FadeOut` - 淡入/淡出
- `SlideInFromBottom/Top/Left/Right` - 滑入动画

#### 5. UI绑定辅助工具
新增 `UIBinder` 工具类，简化UI组件引用获取：

```csharp
// 查找组件
var button = transform.FindComponent<Button>("Panel/Button");
var text = transform.FindComponentRecursive<Text>("Title");

// 快捷绑定事件
transform.BindButton("CloseBtn", () => Close());
transform.BindToggle("SoundToggle", isOn => SetSound(isOn));
transform.BindSlider("VolumeSlider", value => SetVolume(value));

// 设置文本和图片
transform.SetText("Title", "欢迎");
transform.SetSprite("Icon", iconSprite);

// 自动绑定（使用特性）
public class MyPanel : UIPanel
{
    [UIBind("CloseBtn")] private Button _closeBtn;
    [UIBind("Title")] private TextMeshProUGUI _title;
    
    protected override void OnCreate()
    {
        UIBinder.AutoBind(this, transform);
    }
}
```

### 新增文件
- `IUIAnimation.cs` - UI动画接口和预定义动画
- `UIBinder.cs` - UI绑定辅助工具

## 🎉 v2.0 - 多界面切换系统 (2024-12-01)

### 新功能：Tab系统

新增了完整的多界面切换系统，支持游戏中主页的多个页签切换功能。

#### 核心组件
- ✅ **UITabPage** - 标签页基类，既可独立使用又可作为容器子页面
- ✅ **UITabContainer** - 标签页容器，使用组合模式管理多个TabPage
- ✅ **UITabButton** - 标签按钮组件，用于切换页面
- ✅ **UITabContainerBuilder** - 构建器，提供流式API简化容器构建

#### 核心特性
- ✅ 组合模式管理父子UI生命周期
- ✅ 子页面可独立使用或作为容器一部分（对子页面无感）
- ✅ 支持索引、类型、名称多种方式切换页面
- ✅ 页面切换事件通知
- ✅ Builder模式简化容器构建

#### 新增文件
- `UITabPage.cs` - 标签页基类
- `UITabContainer.cs` - 标签页容器
- `UITabButton.cs` - 标签按钮组件
- `UITabContainerBuilder.cs` - 容器构建器
- `TabSystemExample.cs` - 完整示例
- `多界面切换系统文档.md` - 详细设计文档
- `Tab系统快速开始.md` - 快速入门指南

#### 使用示例
```csharp
// 创建容器
var container = Instantiate(containerPrefab);

// 添加页面
container.AddPage(Instantiate(homePagePrefab));
container.AddPage(Instantiate(inventoryPagePrefab));
container.AddPage(Instantiate(shopPagePrefab));

// 切换页面
container.SwitchPage(0);                    // 通过索引
container.SwitchPage<InventoryPage>();      // 通过类型
container.SwitchPage("ShopPage");           // 通过名称
```

## 🎯 新的依赖关系

### 父组件 → 子组件

```csharp
public class InventoryPanel : UIPanel
{
    // ✅ 父组件持有子组件引用
    private List<ItemSlotComponent> _slots = new List<ItemSlotComponent>();
    
    protected override void OnCreate()
    {
        // ✅ 父组件创建和管理子组件
        for (int i = 0; i < 20; i++)
        {
            var slot = Instantiate(slotPrefab);
            ComponentManager.RegisterComponent(slot);
            _slots.Add(slot);  // 持有引用
        }
    }
    
    // ✅ 父组件可以直接调用子组件方法
    public void SelectSlot(int index)
    {
        _slots[index].SetSelected(true);
        _slots[index].Show();
    }
}
```

### 子组件 → 事件总线

```csharp
public class ItemSlotComponent : UIComponent
{
    // ✅ 子组件不持有父组件引用
    // ❌ 不存在 ParentView 属性
    
    private void OnClicked()
    {
        // ✅ 发送全局事件（不知道谁在监听）
        SendEvent(new ItemClickedEvent { ... });
    }
}
```

### 父组件接收事件

```csharp
public class InventoryPanel : UIPanel
{
    protected override void OnCreate()
    {
        // ✅ 订阅全局事件
        xFrameEventBus.SubscribeTo<UIComponentEventWrapper<ItemClickedEvent>>(
            OnItemClicked);
    }
    
    private void OnItemClicked(ref UIComponentEventWrapper<ItemClickedEvent> evt)
    {
        // ✅ 通过ComponentId识别是哪个子组件
        var slot = ComponentManager.GetComponent<ItemSlotComponent>(evt.ComponentId);
        
        // ✅ 或者直接使用SourceComponent
        var slot2 = evt.SourceComponent as ItemSlotComponent;
        
        // ✅ 直接调用子组件方法
        slot.SetSelected(true);
    }
}
```

## 🌟 核心优势

### 1. 符合设计模式
- ✅ 遵循组合模式（Composite Pattern）
- ✅ 父节点持有子节点引用
- ✅ 子节点不持有父节点引用

### 2. 降低耦合
- ✅ 子组件不依赖具体的父组件类型
- ✅ 子组件可以在任何父组件中复用
- ✅ 子组件保持独立性和纯粹性

### 3. 清晰的控制权
- ✅ 父组件拥有控制权
- ✅ 父组件管理子组件的创建、销毁、生命周期
- ✅ 父组件协调多个子组件之间的交互

### 4. 单向数据流
- ✅ 父 → 子：直接调用方法
- ✅ 子 → 父：通过事件通知
- ✅ 数据流向清晰明确

## 📚 迁移指南

如果你有基于旧版本的代码，需要做以下调整：

### 1. 移除子组件中的父组件引用

```csharp
// ❌ 旧代码
public class MyComponent : UIComponent
{
    private void DoSomething()
    {
        if (ParentView is MyPanel panel)
        {
            panel.HandleEvent();
        }
    }
}

// ✅ 新代码
public class MyComponent : UIComponent
{
    private void DoSomething()
    {
        SendEvent(new MyComponentEvent { ... });
    }
}
```

### 2. 修改事件发送方式

```csharp
// ❌ 旧代码
SendEventToParent(new ItemClickedEvent { ... });

// ✅ 新代码
SendEvent(new ItemClickedEvent { ... });
```

### 3. 父组件持有子组件引用

```csharp
// ✅ 新代码：父组件持有引用
public class MyPanel : UIPanel
{
    private List<MyComponent> _components = new List<MyComponent>();
    
    protected override void OnCreate()
    {
        for (int i = 0; i < 10; i++)
        {
            var comp = Instantiate(prefab);
            ComponentManager.RegisterComponent(comp);
            _components.Add(comp);  // 持有引用
        }
    }
    
    // 可以直接操作
    public void HideAll()
    {
        foreach (var comp in _components)
        {
            comp.Hide();
        }
    }
}
```
